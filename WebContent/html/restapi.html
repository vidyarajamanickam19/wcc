Website Quality Checker
Crawl report

This API is used to generate a report for the links entered by the user. The report consist of a counters for each type such as html file, download file or email address. The report also contains a list of URLs which are invalid. The list is segregated into three list containing links to either html file, download file or an email address.

Resource URI http://host:port/crawlreport?website=http://website-to-crawl.com&crawllevel=1
HTTP Method GET
Message Format JSON
Sample Request http://localhost:8080/crawlreport?website=http://16.181.235.85:8080&crawllevel=1
